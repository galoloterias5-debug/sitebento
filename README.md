# Bento Açaíteria — Site (GitHub Pages)

## Publicar
1. Crie um repositório no GitHub (Public).
2. Faça upload de **todos** os arquivos desta pasta na raiz do repositório.
3. Vá em **Settings → Pages**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Salve e aguarde o link aparecer.

## Dica de cache
Se não atualizar no celular, force recarregar (Ctrl+F5) ou limpe cache do navegador.

## Config
- Horário de funcionamento: `config.js` → `window.CONFIG.HOURS`
